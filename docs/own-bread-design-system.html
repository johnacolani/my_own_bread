Principles

Decisions for Own Bread: clarity, timing, and calm guidance — not generic notes.

  • Warm structure — Primary teal #6FA8A1 for structure, links, and focus; coral #D98C7A for primary actions (Start bake, Save, Finish). Purple and gold stay as soft accents in documentation swatches; the Flutter theme centers on teal + coral.
  • Surfaces — Page background #F5F2EB, cards #FFFCF7 — flour-and-crumb warmth, not stark white.
  • Time is content — Proof windows, bake time, and per-step minutes are first-class; UI should surface duration without clutter (chips, step subtitles).
  • Type — Roboto via Flutter defaults; scale matches AppTypography (heading 28 → label 12).
  • Spacing — AppSpacing uses a tight 4-based scale (4–32px) for consistent padding in forms and lists.
