# Own Bread Design System

This starter translates the uploaded HTML design system into a Flutter-friendly theme layer.

## Color roles

- Teal `#6FA8A1`: structure, calm guidance, progress
- Coral `#D98C7A`: primary action
- Purple `#7B6F9D`: secondary accents
- Gold `#C9A96E`: warmth, completion, reward
- Background `#F5F2EB`: soft baking surface
- Dark `#1A1917`: warm charcoal

## Tokens

- `AppColors`
- `AppSpacing`
- `AppRadius`
- `AppTypography`
- `AppTheme`

## Component direction

- Rounded cards
- Warm surfaces
- Clear CTA hierarchy
- Calm step-by-step layouts
